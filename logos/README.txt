COMO ADICIONAR OS LOGOS DAS CASAS DE APOSTA
============================================

Coloque cada logo nesta pasta (logos/) com EXATAMENTE o nome de arquivo abaixo.
- Formato: PNG (de preferência com fundo transparente). Pode ser também .jpg/.svg,
  mas o código procura ".png" — se usar outro formato, me avise que eu ajusto.
- Tamanho sugerido: altura ~60px (ele é exibido pequeno na lista).
- Se faltar algum arquivo, aquela casa simplesmente mostra o NOME colorido (não quebra nada).
